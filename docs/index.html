Project March 2025

debug-gym

A text-based environment for interactive debugging that enables Large Language Models (LLMs) to interactively explore codebases and gather relevant information for debugging tasks. The environment provides preset tools like Python debugger (pdb) to facilitate LLM-based agents' interactive debugging capabilities.
